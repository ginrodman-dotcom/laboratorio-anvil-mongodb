# Laboratorio CRUD con Anvil y MongoDB

## Descripción
Este repositorio contiene el desarrollo de un laboratorio académico donde se implementa
un sistema CRUD (Crear, Leer, Actualizar y Eliminar) utilizando Python, MongoDB y la
plataforma Anvil.

La aplicación permite gestionar productos de una tienda musical mediante una interfaz
gráfica web conectada a una base de datos NoSQL.

---

## Tecnologías utilizadas
- Python
- MongoDB Atlas
- Anvil
- PyMongo

---

## Funcionalidades principales
- Visualización de productos
- Búsqueda y filtrado por nombre y género
- Ordenamiento por precio
- Agregar nuevos productos
- Actualizar productos existentes
- Eliminar productos
- Interfaz gráfica con RepeatingPanel

---

## Estructura del proyecto
- `ServerModule1.py`: conexión y lógica de base de datos
- `A_generar_datos.py`: generación de datos iniciales
- `B_limpiar_bd.py`: limpieza de la base de datos
- `C_crud_completo.py`: operaciones CRUD
- `anvil_code/`: código de los formularios de Anvil

---

## Ejecución y publicación
La aplicación fue ejecutada y verificada mediante el botón **Run** en Anvil y posteriormente
publicada utilizando la opción **Publish**, generando una URL pública para su acceso.
